# Python Basics

A collection of beginner Python exercises covering core fundamentals.

## Contents

- **practice.py** — Practice exercises covering variables, data types, input/output, and operators (name/age printing, greeting the user, adding numbers, Celsius-to-Fahrenheit conversion, rectangle area).
- **calculator.py** — A simple command-line calculator that takes two numbers and an operator (`+`, `-`, `*`, `/`) and prints the result, using `if`/`elif`/`else` statements.

## Topics covered

- Variables
- Data types (int, float, string, boolean)
- Input & output
- Operators
- Comments
- Conditional statements (`if` / `elif` / `else`)

## How to run

```bash
python calculator.py
python practice.py
```

## Example: calculator.py

```
Enter first number: 10
Enter second number: 5
Choose operation (+, -, *, /): +
Answer: 15.0
```
